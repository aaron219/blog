üåµTo make DockerCE can access internet through proxy server.

Linux: Ubuntu 20.04
Docker: 19.03.12

üå¥Setup proxy on DockerCE

1. Create a folder

  • In following location
mkdir /etc/systemd/system/docker.service.d

2. Create a configure file

  • The file name: http-proxy.conf
vi http-proxy.conf
  • Add the following value
[Service]
Environment="HTTP_PROXY=<ip>:<port>"
Environment="HTTPS_PROXY=<ip>:<port>"
Environment="NO_PROXY=localhost,127.0.0.1,localaddress,.localdomain.com"

3. Restart the daemon and DockerCE

sudo systemctl daemon-reload
sudo systemctl restart docker
