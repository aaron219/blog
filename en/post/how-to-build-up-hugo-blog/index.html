üêπInstall Hugo

1. Verify git

git --version

2. Download Hugo and install

i. Windonws

  • Goto https://github.com/gohugoio/hugo/releases
  • Download the last version of hugo for windows (hugo_0.XX.X_Windows-64bit.zip)
  • Unzip the exe file to C:\hugo
  • In CMD, Enter following:
pathman /au c:\hugo

ii. Ubuntu

  • Install with apt-get
sudo apt-get install hugo

iii. CentOS

  • download to this path: /etc/yum.repos.d/
wget https://copr.fedorainfracloud.org/coprs/daftaupe/hugo/repo/epel-7/daftaupe-hugo-epel-7.repo -O hugo.repo
  • Install with yum
yum install hugo

3. Check Hugo version

hugo version

üê∏Configure Hugo

1. Create a site

hugo new site <sitename>

2.1. Add theme (git clone)

git clone https://github.com/syui/hugo-theme-air themes/air

2.2. Add theme (git submodule)

git init
git submodule add https://github.com/AmazingRise/hugo-theme-diary themes/diary

3. Apply theme for modify configure file

  • configure file name and path: <sitename>/config.toml
  • modify the theme value
theme = diary
  • add some prosenal information in configure file, e.g. blog’s name, url, email

3.1. Replace configure file

  • some theme need a special config file
cp  theme/<themename>/exampleSite/config.toml ~/<sitename>

4. Replace static and layouts folder

  • copy static and layouts folder to root
cp -R theme/<themename>/static ~/<sitename>
cp -R theme/<themename>/layouts ~/<sitename>

5. Start Hugo

hugo server --bind=0.0.0.0 --baseURL=http://<IP> --port=1313

5.1 Start Hugo and generate static web page file

hugo server --bind=0.0.0.0 --baseURL=http://<IP> --port=1313 -D

END
