How to Build Up Hexo Blog and deploy to Github

üçÖEasy and sample build up Hexo Blog and delpoy to Github. Just a note for this blog build up.

git: 2.18.4-2.el8_2
git: 2.24.1.windows.2
hexo: 5.2.0
hexo-cli: 4.2.0
CentOS: 8.2.2004
Windows: 10 1909

üçêInstall

1. Verify git, node and npm

git --version
node -v
npm -v

2. Upgrade npm version (option)

npm install npm -g

3. Install hexo-cli hexo-admin

npm install hexo-cli hexo-admin -g

4. Check hexo

hexo version

üçëConfigure Hexo

1. Create a site

hexo init <sitename>

2. Add theme (git clone)

git clone https://github.com/next-theme/hexo-theme-next themes/next

3. Apply theme for modify configure file

  • configure file name and path: <user>/<sitename>/_config.yml

  • modify the theme value

theme: next

4. Install tools for hexo insert picture

npm install hexo-asset-image --save

5. Apply insert picture funcation for modify configure file

  • configure file name and path: <user>/<sitename>/_config.yml
  • modify the post asset value
post_asset_folder: true

6. Auto install compatible updates to vulnerable dependencies

npm audit fix

7. Start hexo server and generate static web page file

hexo clean
hexo s -g

8. Check web page

  • Hexo page: http://localhost:4000/

  • Hexo admin page: http://localhost:4000/admin/

üçÜDeploy to Github

1. Install hexo’s deploy tool for github

npm install hexo-deployer-git --save

2. Setup deploy plugin

  • configure file name and path: <sitename>/_config.yml
  • add new value
deploy:
  type: git
  repository: git@github.com:<username>/<repository_name>.git
  branch: gh-pages

üçáPost setting

1. Setup the save path of post

  • configure file name and path: <sitename>/_config.yml
  • modify the new post value
new_post_name: :year/:title.md

2. Configure post template

  • configure file name and path: <sitename>/scaffolds/post.md
  • modify the permalink value
permalink: {{ title }}

üçâFlowchart plugin

1. Install flowchart plugin

npm install --save hexo-filter-flowchart

2. Setup flowchart plugin

  • configure file name and path: <sitename>/_config.yml
  • add new value
flowchart:
  # raphael:   # optional, the source url of raphael.js
  # flowchart: # optional, the source url of flowchart.js
  options: # options used for `drawSVG`

üçåMermaid plugin

1. Install mermaid plugin

npm install hexo-filter-mermaid-diagrams

2.1. Setup mermaid plugin

  • configure file name and path: <sitename>/_config.yml
  • add new value
# mermaid chart
mermaid: ## mermaid url https://github.com/knsv/mermaid
  enable: true  # default true
  version: "7.1.2" # default v7.1.2
  options:  # find more api options from https://github.com/knsv/mermaid/blob/master/src/mermaidAPI.js
    #startOnload: true  // default true
  • configure file name and path: <sitename>/themes/next/layout/_partials/footer.swig
  • add new value
{% if theme.mermaid.enable %}
  <script src='https://unpkg.com/mermaid@{{ theme.mermaid.version }}/dist/mermaid.min.js'></script>
  <script>
    if (window.mermaid) {
      mermaid.initialize({{ JSON.stringify(theme.mermaid.options) }});
    }
  </script>
{% endif %}

2.2. Setup mermaid plugin (for next theme)

  • configure file name and path: //themes/next/_config.yml
  • find the # mermaid chart
  • modify to the following value
# mermaid chart
mermaid: ## mermaid url https://github.com/knsv/mermaid
  enable: true  # default true
  version: "7.1.2" # default v7.1.2
  options:  # find more api options from https://github.com/knsv/mermaid/blob/master/src/mermaidAPI.js
    #startOnload: true  // default true

END
