git: 2.18.4-2.el8_2
git: 2.24.1.windows.2
hugo: v0.75.1

‚òÄÔ∏èGithub setting

1. Create a Github account

  • Goto Github for sign up a account

2. Create Personal access tokens

  • Create a personal access tokens for deploy github pages

    i. In main page, Goto Settings

    ii. Select Developer settings, then select Personal access tokens

    iii. Click Generate new token button

    iv. Type Note and select repo:status & public_repo only, then Click Update token

    v. After that the token will be generate and display in the page

    vi. Backup the token, beacause after that it will not appear again

3. Create repository

  • Create repository for code only

    i. Goto Repositories page

    ii. Click New button

    iii. Type repository name and select Private, then click Create repository

  • Create repository for Github pages (blog page)

    i. Goto Repositories page

    ii. Click New button

    iii. Type repository name and select Public, then click Create repository

‚òÅÔ∏èBuild up hugo blog

Follow “How to Build Up Hugo Blog” for build up the blog

‚ùÑÔ∏èCreate github actions running file

1. Create folder

  • Create a folder and sub-folder in hugo blog root directory
  • Name of folder MUST be “.github” and sub-folder is “workflows”
mkdir -p .github/workflows

2. Create a execute file

  • Locale: .github/workflows
  • Name of file call: main.yml
name: Github-Pages  ## execute name

on:
  push:
    branches:
      - main  ## when push the main branch this file will be execute

jobs:
  deploy:
    runs-on: ubuntu-18.04  ## deploy the Hugo blog use ubuntu-18.04
    steps:
      - name: Checkout
        uses: actions/checkout@v1
        ## use github project actions/checkout (https://github.com/actions/checkout)
        with:
          submodules: true

      - name: Setup Hugo
        uses: peaceiris/actions-hugo@v2
        ## use github project peaceiris/actions-hugo (https://github.com/peaceiris/actions-hugo)
        ## this project will setup a environment for deploy Hugo
        with:
          hugo-version: '0.75.1'
          extended: true

      - name: Build
        run: hugo --gc --minify --cleanDestinationDir
        ## run hugo command

      - name: Deploy
        uses: peaceiris/actions-gh-pages@v3
        ## use github project peaceiris/actions-gh-pages (https://github.com/peaceiris/actions-gh-pages)
        ## this project will deploy static web page to github pages
        with:
          personal_token: ${{ secrets.PERSONAL_TOKEN }}
          external_repository: [username]/[repository_name]  ## publish to github pages's repository
          publish_branch: gh-pages  ## publish to gh-pages branch
          force_orphan: true  ## force upload static web page

‚ö°Deploy Github Pages (Blog page)

1. Add Hugo blog code to Git loacl repository

  • In hugo blog root directory
git init
git add .
git commit -m "first commit"

2. Connect to remote repository

git remote add origin https://github.com/[username]/[repository_name].git

3. Push Hugo blog code to Github

  • Push the code to Github
git push -u origin main

4. Deploy Github Pages

  • When push is completed, the Github Actions will auto deploy the Github Pages

  • Can check the deploy status on the code repository’s Actions tab

    • In Actions, wil show all the task

    • Click the task name can see the detail
