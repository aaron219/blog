架設Hugo的部落格

 發表於 September 29, 2020  |  1 分鐘  |  123 個字  |  Aaron  • 其他語言: English

🐰簡易安裝架設Hugo的部落格，順便記錄架設部落格的過程。

git: 2.18.4-2.el8_2
git: 2.24.1.windows.2
hugo: v0.75.1
CentOS: 8.2.2004
Windows: 10 1909

🐹安裝Hugo

1. 檢查git是否有安裝

git --version

2. 下載及安裝Hugo

i. Windonws

  • 下載網址 https://github.com/gohugoio/hugo/releases
  • 下載最新版本hugo (hugo_0.XX.X_Windows-64bit.zip)
  • 把文件解壓到 C:\hugo
  • 開啓CMD, 輸入以下指令:
pathman /au c:\hugo"

ii. Ubuntu

  • 使用apt-get指令安裝
sudo apt-get install hugo

iii. CentOS

  • 把repo文件下載到: /etc/yum.repos.d/
wget https://copr.fedorainfracloud.org/coprs/daftaupe/hugo/repo/epel-7/daftaupe-hugo-epel-7.repo -O hugo.repo
  • 使用yum指令安裝
yum install hugo

3. 檢查hugo版本

hugo version

🐸設定Hugo

1. 建立部落格網站

hugo new site <sitename>

2.1. 新增主題 (git clone)

git clone https://github.com/syui/hugo-theme-air themes/air

2.2. 新增主題 (git submodule)

git init
git submodule add https://github.com/AmazingRise/hugo-theme-diary themes/diary

3. 應用主題及修改配置文件

  • 配置文件路徑及名稱: <sitename>/config.toml
  • 修改主題設定
theme = diary
  • 在配置文件上新增一些私人資料，如部落格名稱、網址、電郵地址等

3.1. 使用主題預設配置文件

  • 有些主題需要一些特別配，故可以使用主題預設的配置文件
cp  theme/<themename>/exampleSite/config.toml ~/<sitename>

4. 使用預設的static及layouts資料夾取代根目錄的同名資料夾取

  • 複製static及layouts兩個資料夾到根目錄
cp -R theme/<themename>/static ~/<sitename>
cp -R theme/<themename>/layouts ~/<sitename>

5. 啓動Hugo

hugo server --bind=0.0.0.0 --baseURL=http://<IP> --port=1313

5.1 啓動Hugo並生成靜態網站文件

hugo server --bind=0.0.0.0 --baseURL=http://<IP> --port=1313 -D

完
