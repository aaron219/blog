使用Github Actions生成Hugo部落格

 發表於 October 9, 2020  |  2 分鐘  |  299 個字  |  Aaron  • 其他語言: English

🌕在Github中利用Github Actions及Hugo自動生成靜態網頁並上載到Github Pages.

git: 2.18.4-2.el8_2
git: 2.24.1.windows.2
hugo: v0.75.1

☀️設定Github

1. 在Github網站注冊帳戶

  • 到 Github 注冊帳戶

2. 設定生成Personal access tokens

  • 生成一個用來上載github pages的personal access tokens

    i. 在用戶主頁面, 選擇 Settings

    ii. 選擇 Developer settings, 之後再選擇 Personal access tokens

    iii. 點擊 Generate new token

    iv. 在 Note 中輸入註解並選擇 repo:status 及 public_repo , 之後點擊 Update token

    v. 之後頁面會出現生成的token (一個字串)

    vi. 如有需要可自行備份token, 因為該token只會出現一次

3. 建立倉庫

  • 建立用作存放代碼的倉庫

    i. 到 Repositories 分頁

    ii. 點擊 New

    iii. 在 repository name 中輸入倉庫名稱並選擇 Private, 之後再點擊 Create repository

  • 建立用作存放Github pages(blog page)的倉庫

    i. 到 Repositories 分頁

    ii. 點擊 New

    iii. 在 repository name 中輸入倉庫名稱並選擇 Public, 之後再點擊 Create repository

☁️建立Hugo部落格

参考 “How to Build Up Hugo Blog” 去建立Hugo部落格

❄️建立Github Actions運行的檔案

1. 建立以下資料夾

  • 在Hugo部落格的根目錄建立一個資料夾及一個子資料夾
  • 資料夾名稱一定要命名為".github", 子資料夾命名為"workflows"
mkdir -p .github/workflows

2. 建立需要運行的檔案

  • 檔案儲存位置: .github/workflows
  • 檔案名稱: main.yml
name: Github-Pages  ## execute name

on:
  push:
    branches:
      - main  ## when push the main branch this file will be execute

jobs:
  deploy:
    runs-on: ubuntu-18.04  ## deploy the Hugo blog use ubuntu-18.04
    steps:
      - name: Checkout
        uses: actions/checkout@v1
        ## use github project actions/checkout (https://github.com/actions/checkout)
        with:
          submodules: true

      - name: Setup Hugo
        uses: peaceiris/actions-hugo@v2
        ## use github project peaceiris/actions-hugo (https://github.com/peaceiris/actions-hugo)
        ## this project will setup a environment for deploy Hugo
        with:
          hugo-version: '0.75.1'
          extended: true

      - name: Build
        run: hugo --gc --minify --cleanDestinationDir
        ## run hugo command

      - name: Deploy
        uses: peaceiris/actions-gh-pages@v3
        ## use github project peaceiris/actions-gh-pages (https://github.com/peaceiris/actions-gh-pages)
        ## this project will deploy static web page to github pages
        with:
          personal_token: ${{ secrets.PERSONAL_TOKEN }}
          external_repository: [username]/[repository_name]  ## publish to github pages's repository
          publish_branch: gh-pages  ## publish to gh-pages branch
          force_orphan: true  ## force upload static web page

⚡生成Blog頁面及上載至Github Pages

1. 把Hugo部落格的代碼加入到Git本地倉庫

  • 在Hugo部落格的根目錄
git init
git add .
git commit -m "first commit"

2. 連結遠端倉庫(Github)

git remote add origin https://github.com/[username]/[repository_name].git

3. 上載Hugo部落格代碼到Github

  • 上載代碼到Github
git push -u origin main

4. 生成及上載至Github Pages

  • 當上載代碼完成時, Github Actions會自動生成及上載Hugo部落格至Github Pages

  • 可於代碼倉庫的Actions分頁查看自動生成及上載的情況

    • 在Actions頁面, 會顯示所有的任務(task)

    • 點擊任務名稱, 可查看任務詳細情況

完
